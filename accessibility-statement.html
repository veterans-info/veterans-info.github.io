Accessibility Statement

Last updated: January 1, 2024

Our Commitment to Accessibility

The Veterans' Preference Guide is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

Veterans' Preference Guide is fully conformant with WCAG 2.1 level AA. Fully conformant means that the content fully conforms to the accessibility standard without any exceptions.

Accessibility Features

We've designed this website with accessibility in mind. Key features include:

  • Clear and consistent navigation structure
  • Keyboard navigation support throughout the site
  • Skip navigation links on all pages
  • Descriptive link text
  • Alternative text for all informative images
  • Sufficient color contrast ratios
  • Resizable text without loss of functionality
  • Properly structured headings
  • ARIA labels and landmarks where appropriate
  • Form labels and error messages
  • Focus indicators for interactive elements
  • Compatibility with screen readers

Technologies

This website is designed to be compatible with the following assistive technologies:

  • Screen readers (JAWS, NVDA, VoiceOver)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation

The website uses the following technologies:

  • HTML5
  • CSS3
  • JavaScript (with progressive enhancement)
  • WAI-ARIA

Assessment Methods

We assessed the accessibility of Veterans' Preference Guide by the following approaches:

  • Self-evaluation using automated tools (WAVE, axe DevTools)
  • Manual keyboard navigation testing
  • Screen reader testing with NVDA
  • Color contrast analysis
  • Mobile device testing

Known Limitations

Despite our best efforts to ensure accessibility of Veterans' Preference Guide, there may be some limitations. Below is a description of known limitations and potential solutions:

  1. PDF documents: Some linked OPM resources may be in PDF format that is not fully accessible. We are working to provide HTML alternatives or summaries where possible.
  2. Third-party content: External links to OPM, VA, and USAJOBS websites are outside our control but are necessary for official information.

Feedback

We welcome your feedback on the accessibility of Veterans' Preference Guide. Please let us know if you encounter accessibility barriers:

  • Email: feedback@veterans-info.github.io
  • We aim to respond to accessibility feedback within 5 business days

Compatibility

This website is designed to be compatible with:

  • Recent versions of Chrome, Firefox, Safari, and Edge browsers
  • Mobile devices running iOS and Android
  • Screen readers including JAWS, NVDA, and VoiceOver

Technical Specifications

Accessibility of Veterans' Preference Guide relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript
  • WAI-ARIA

These technologies are relied upon for conformance with the accessibility standards used.
