Our Commitment to Accessibility

We are committed to making the Veterans Info website accessible to all users, regardless of ability. We strive to ensure that our website is usable and understandable by people with diverse needs, including those who use assistive technologies.

Accessibility Standards

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, where feasible for a static site. These guidelines explain how to make web content more accessible for people with disabilities.

Measures Taken to Support Accessibility

  • Semantic HTML: We use semantic HTML to provide a clear and logical structure for screen readers and other assistive technologies.
  • Keyboard Navigation: All interactive elements are designed to be navigable and operable using only a keyboard.
  • Color Contrast: We ensure sufficient color contrast between text and background to improve readability for users with visual impairments.
  • Alternative Text: All meaningful images will include descriptive alternative text.
  • Responsive Design: The website is designed to be responsive and accessible across various devices and screen sizes.
  • Skip Links: A "skip to main content" link is provided for easy navigation past repetitive header content.

Privacy-Respecting Analytics

This website uses Plausible Analytics for basic, privacy-friendly website analytics. Plausible Analytics is a lightweight, open-source, and GDPR-compliant web analytics tool that does not use cookies and does not collect any personal data or personally identifiable information (PII). It helps us understand general website traffic patterns (e.g., page views, popular content) without tracking individual users.

Known Limitations

While we strive for full accessibility, there may be some areas that are still undergoing improvement. We are continuously working to enhance the accessibility and usability of our website.

Feedback

We welcome your feedback on the accessibility of the Veterans Info website. If you encounter any accessibility barriers or have suggestions for improvement, please contact us. Your input helps us make our website better for everyone.
